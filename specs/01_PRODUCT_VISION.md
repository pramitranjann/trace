# 01 — Product Vision

## Vision

Trace makes learning to read and write feel active, playful, and memorable.

It is built around a simple idea:

Learners should not only see a letter.
They should hear it, move with it, and produce it.

## Current MVP

The MVP focuses on English capital letters:

- A
- B
- C
- D
- E

This is not because English is the final product direction.

English is chosen because:

- data is easier to test
- users are easier to find
- recognition is easier
- the team can validate the learning model faster

## Long-Term Direction

The platform should eventually support many scripts:

- Hindi
- Tamil
- Bengali
- Japanese
- Korean
- Arabic
- Chinese
- English

The core product model remains the same:

- See
- Hear
- Produce

Only the content changes.

## Product Positioning

Internal positioning:

A multimodal literacy platform.

External MVP positioning:

A playful app that helps kids learn letters by seeing, hearing, tracing, and writing them.

## What Makes It Different?

Most literacy apps focus on recognition.

Trace connects:

- visual recognition
- audio recognition
- motor memory
- handwriting
- real-world text

This makes the world itself part of the learning material.

## North Star

Make learning to write feel as engaging, intuitive, and rewarding as learning to speak.
