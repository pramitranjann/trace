# PROJECT_CONTEXT.md

## Project

Trace is a fast MVP for a multimodal literacy platform.

The product teaches letters through three modes:

- See
- Hear
- Produce

It is meant to move beyond static worksheets by making repetition interactive, playful, physical, and real-world aware.

## Founder Context

Pramit is a UX Design student at SCAD with experience building playful camera-based prototypes, including browser-based MediaPipe projects.

Trace should feel like a portfolio-quality project and potential startup exploration.

## Design Intent

The MVP should not look like a generic ed-tech dashboard.

It should feel:

- playful
- bright
- childlike
- tactile
- brandable
- animated
- fast to understand

Temporary UI is fine, but it should not block future UI import.

## Existing Relevant Experience

Relevant project patterns from prior work:

- browser-based MediaPipe interaction
- playful gesture-based game loops
- Vercel-hosted prototypes
- strong UX-first thinking
- fast build / fast test workflow

## Product Constraint

The goal is not to build a full language app.

The goal is to test whether See, Hear, and Produce can make literacy repetition more engaging than a traditional workbook.
